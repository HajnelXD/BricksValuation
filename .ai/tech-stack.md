Frontend:
- Vue 3
- TypeScript 5 dla statycznego typowania kodu i lepszego wsparcia IDE
- Tailwind 4 pozwala na wygodne stylowanie aplikacji
- Shadcn/vue zapewnia bibliotekę dostępnych komponentów React, na których oprzemy UI
- Pinia do zarzadzania store’m
- i18n do przechowywania tekstów
- vite
- eslint
- prettier
- yarn

Backend:
- Python
- Django jako główny framework backendowy
- django-rest-framework do udostępniania interfejsow API

Baza danych:
- PostgreSQL

Środowisko uruchomieniowe:
- Docker
- docker compose do uruchamiania i łączenia wszystkich serwisów (Baza danych, frontend i backend)


CI/CD i Hosting:
- Github Actions do tworzenia pipeline’ów CI/CD
